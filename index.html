Recommending the right instance size for a customer : If your database and workloads require a smaller instance size, Our recommendations will help you to scale down and reduce your cost. Conversely, if your workload increases then we recommend you to scale up at that precise moment.

Application Tuning

JATT, a HotSpot autotuner, has been designed specifically to autotune the Java Virtual Machine (JVM). JATT organizes the JVM flag into groups and finds the best combination of flags that would result in the best performance. The rationale for introducing different flag group is to reduce the search space and avoid invalid flag combinations.

Data Security & Privacy

We don’t read your data or any data tables of your database. We only read the current database configurations and metrics generated during query execution. Fault resistant : Any configurations that our system recommends undergoes a strict validation which ensures your database never encounter any failure.

Mode of operations

The customer only receives the recommended configurations, but we don’t apply on the database. This is only for trial purpose and for experiments by the customer. We don’t charge for this service and we gather feedback for improvements.

Performance Improvement

The client provides us, the worst performing query for his database. We formulate a load test where such queries are fired in very high frequency. We observe the performance of client database for this load test before and after our recommendations.
